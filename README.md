📌 Project Overview

This project is a web-based Task Management Dashboard deployed on AWS using an EC2 instance.
It demonstrates cloud deployment, Linux server configuration, and Flask-based web development.

☁️ Cloud Architecture
AWS EC2 instance (Amazon Linux)
Flask web framework
Public IPv4 access
Security Group allowing HTTP (port 80)
VPC default networking


🛠️ Technologies Used
AWS EC2
Flask (Python)
HTML / CSS
Linux (Amazon Linux 2023)
Git & GitHub


⚙️ Deployment Steps
1. EC2 Setup
Launched EC2 instance
Configured key pair for SSH access

2. Server Setup
Installed Python 3
Installed Flask using pip
Created application file app.py

3. Application Deployment
Built a Flask web server
Served an HTML dashboard using templates
Bound application to 0.0.0.0:80

4. Networking Configuration
Configured Security Group (HTTP port 80 open)
Verified Internet Gateway and routing

5. Testing
Verified local server using curl
Accessed the application via public IP
